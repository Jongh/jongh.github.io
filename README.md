---

## 소개

시스템 운영과 아키텍처 설계를 아는 풀스택 개발자의 개인 포트폴리오 페이지입니다.
자기소개·업무 이력·주요 업무와 함께, 특정 언어/프레임워크가 아닌 **보편적 AI 활용 방법론**을
역량으로 소개합니다.

소개하는 AI 활용 방법론 7가지:

1. 컨텍스트 엔지니어링
2. 작업 분해
3. 역할 분업 (오케스트레이션)
4. 검증·승인 게이트
5. 검증 루프
6. 의사결정 기록
7. 재사용 자산화

## 페이지 구성

| 섹션 | 내용 |
|------|------|
| 히어로 | 이름·직함·CTA (주요 업무 보기 · 이력서 다운로드) |
| 숫자로 보는 경력 | 경력·프로젝트·도메인·운영 기간 지표 |
| 자기소개 | 배경·철학·차별점 |
| 업무 이력 | 경력 타임라인 |
| 도메인·산업 경험 | 다룬 산업 도메인 맵 |
| 주요 업무 | 대표 프로젝트 |
| AI 활용 방법론 | 7가지 협업 방법론 + 실증 |
| AI 협업 방식 | 설계→구현→검토→기록 흐름 |
| 연락처 | GitHub · 이메일 · 이력서 다운로드 |

## 기술 구성

| 파일 | 역할 |
|------|------|
| `index.html` | 단일 페이지 (9개 섹션 + sticky 네비게이션) |
| `style.css` | 베이지 테마, CSS Grid 카드 레이아웃, 반응형 |
| `main.js` | 스크롤 fade-in (IntersectionObserver) |
| `favicon.png` | 파비콘 |
| `assets/image/` | 프로필·OG 썸네일 이미지 |
| `assets/resume/` | 이력서 PDF (웹 공개용) |
| `.nojekyll` | Jekyll 빌드 비활성화 |

빌드 도구 없음 · 외부 라이브러리 없음 · 순수 HTML/CSS/JS
